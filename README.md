# back-justifica-ai

Para criar um ambiente virtual, execute os seguintes comandos:
```bash
  python -m venv .venv
```

Ativar ambiente virtual no Windows:
```bash
  .venv\Scripts\activate
```

Instalar dependências:
```bash
  pip install -r requirements.txt
```
